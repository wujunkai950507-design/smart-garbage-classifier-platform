# 智慧垃圾分類 AI 平台（Smart Garbage Classifier Platform）

## 一、作品簡介

在日常生活中，許多人對垃圾分類規則並不熟悉，像是「牙膏、紙餐盒、髒掉的紙類」等介於回收與一般垃圾之間的物品，常常不知道要丟哪一類。本作品設計一個「智慧垃圾分類平台」，讓使用者輸入垃圾描述後，系統即可給出對應的分類建議，並記錄歷史資料，協助培養正確的垃圾分類習慣。

本平台未來可與實體「智慧垃圾桶」整合，讓智慧垃圾桶依照系統回傳的分類結果，自動開啟對應的垃圾桶蓋，結合 IoT 與 AI 應用。

---

## 二、系統架構概念

系統主要分成三個部分：

1. **前端網頁（Frontend）**
   - 以 HTML / CSS / JavaScript 實作簡單介面
   - 使用者可以輸入垃圾描述（例如：「用完的牙膏」、「喝完的寶特瓶」）
   - 顯示分類結果與歷史紀錄

2. **分類邏輯（Classification Logic）**
   - 目前版本以 JavaScript 實作簡單規則與關鍵字比對
   - 可依照關鍵字（如「瓶」、「鋁罐」、「電池」、「食物」等）對應到：
     - 一般垃圾
     - 資源回收
     - 廚餘
     - 有害垃圾
   - 未來可替換成雲端 AI 模型 / LLM / Make.com Flow 等方式

3. **IoT 延伸應用（Future IoT Extension）**
   - 可提供一個簡單 API 或代碼輸出（例如：1~4）
   - 讓智慧垃圾桶的微控制器（如 BMduino / ESP32）讀取分類代碼
   - 控制伺服馬達或繼電器，開啟指定垃圾桶蓋

## 四、主要功能說明

1. **垃圾描述輸入**
   - 使用者輸入垃圾名稱或描述文字
   - 例如：「喝完的飲料杯」、「用完的牙膏」、「便當吃剩的菜」

2. **分類結果顯示**
   - 系統根據關鍵字與簡單規則，回傳一個分類建議
   - 例如：
     - 「寶特瓶」→ 資源回收
     - 「紙餐盒（髒）」→ 一般垃圾
     - 「蔬菜、剩飯」→ 廚餘
     - 「電池、燈泡」→ 有害垃圾

3. **歷史紀錄與統計**
   - 前端暫存使用者每次的分類結果
   - 顯示成一個簡單的列表，方便觀察近期分類狀況
   - 可作為未來擴充成圖表或後端資料庫的基礎
